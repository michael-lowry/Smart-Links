# Smart Links
For documentation, see the [Smart Links spec](https://github.com/michael-lowry/Smart-Links/wiki/Smart-Links).

## Files in this directory
* dump_clipboard.ps1 — PowerShell script to dump the contents of the clipboard on Windows.
* dump_clipboard.swift — Swift script to dump the contents of the clipboard on macOS.
* smart_links_demo.html — Web page to demonstrate SLM (Smart Links) levels 0–2.
