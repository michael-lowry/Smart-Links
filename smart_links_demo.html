SLM “Smart Links” demo

This page demonstrates Structured Link Metadata, also known as Smart Links.

SLM level
SLM Level 0 — semantic anchors

Enter URL, anchor text, and optional description. Optionally add system/type/icon ref fields for the Level 2 capsule. The rendered Smart Link supports one-click copy and a right-click menu. Enable hovercard to preview the capsule.


SLM Level 1 — Copy-aware links
The Copy Smart Link action inserts into the clipboard representations of the link in both plain text and HTML, so links can be pasted consistently into email, chat, and documentation.
